# [FastForward] Архитектура Компьютера и Операционные Системы

Данный курс посвящен Архитектуре Компьютера X86_64 и Операционной Системе Linux :)

В рамках курса вам предстоит пройти 14 недель. Каждую неделю планируется лекция, семинар и 5 заданий. 

Дедлайн домашних заданий - 3 недели после соответствующего семинара. После дедлайна в течение 2 недель оценка за задачу плавно убывает до 50%.

С инструкцией по настройке окружения можно ознакомиться [здесь](SETUP.md)

## Примерный план

1) X86_64
2) Язык Си. Философия, синтаксис, линковка, кодировки, куча
3) Файлы, ИО
4) ФС, права, пути
5) Процессы, Сигналы
6) Pipe, MMAP, виртуальная память
7) Время, Локали
8) Потоки, атомарности
9) Сети
10) Docker, vm, cgroup
11) Безопасность, криптография, PKI
