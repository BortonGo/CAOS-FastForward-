### Product

На вход подаётся последовательность целых чисел типа `int64`, заканчивающаяся на `-1`. 

Необходимо вывести произведение всех чисел кроме последней `-1`.

Гарантируется что последовательность содержит хотя бы два числа.

Переполнения возникнуть не может. Результат помещается в `int64`.

Для умножения вам поможет инструкция `imul`, документацию к которой вы можете найти в [официальной документации от интел (заблокировано из РФ)](https://www.intel.com/content/www/us/en/developer/articles/technical/intel-sdm.html), [сохраненной копии](https://disk.yandex.ru/i/ppjYD9rcb7v4vQ)  или на [удобном сайте](https://www.felixcloutier.com/x86/).

Для реализации циклов не забудьте про метки и условные переходы `jmp`, `jz`, `jnz` итд. 

### Пример

#### Input

```
2 3 4 -1
```

#### Output

```
24
```
