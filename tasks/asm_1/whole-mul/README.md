### Product

На вход подаётся последовательность целых чисел типа `int64`, заканчивающаяся на `-1`.

Необходимо вывести произведение всех чисел (не считая `-1`).

#### Формат ввода:
```
a1 a2 a3 ... -1
```

#### Формат вывода:
Одно число — произведение всех элементов.

#### Пример:
```
Вход:
2 3 4 -1

Выход:
24
```
