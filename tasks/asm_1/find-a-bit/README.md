### Find a bit!

На стандартном потоке ввода задается одно целочисленное число
$number$ типа `int64`, его можно считать c помощью функции `readi64` (она находится в `simpleio_x86_64.S` файле). 

Это число можно представить в двоичном виде. Надо найти расстояние до самого дальнего бита этого числа со значением `1`! Также, посчитать $\lfloor log_2(number) \rfloor$.

Вам надо просто вывести в стандартный вывод с помощью метода `writei64`!

Как правильно вывести, смотрите в тестах.
