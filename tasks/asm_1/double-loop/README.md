### Double Loop

На вход подаётся число N — количество строк. Далее идут N строк, каждая из которых содержит последовательность целых чисел типа `int64`, заканчивающуюся на `-1`.

Для каждой строки необходимо вывести максимальное число (не считая `-1`).

#### Формат ввода:
```
N
a1 a2 ... -1
b1 b2 ... -1
...
```

#### Формат вывода:
N чисел — максимум каждой строки.

#### Пример:
```
Вход:
2
1 5 3 -1
10 2 8 -1

Выход:
5
10
```
