### Double Loop

На вход подаётся число N — количество строк. Далее идут N строк, каждая из которых содержит последовательность целых чисел типа `int64`, заканчивающуюся на `-1`.

Для каждой строки необходимо вывести максимальное число (без учета последней `-1`).

### Пример

#### Input

```
2
1 5 3 -1
10 2 8 -1
```

#### Output
```
5
10
```
