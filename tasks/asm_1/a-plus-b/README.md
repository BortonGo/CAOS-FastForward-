### A-plus-B

На стандартном потоке ввода задается два целых числа
типа `int64`, его можно считать с помощью функции `readi64` (она находится в `simpleio_x86_64.S` файле). 

Вам надо посчитать их сумму и вывести в стандартный вывод с помощью метода `writei64` (там же).

В конце программы не забудьте вызвать `call finish`

Вам будут полезны инструкции `add` и `mov`, документацию к которым вы можете найти в [официальной документации от интел (заблокировано из РФ)](https://www.intel.com/content/www/us/en/developer/articles/technical/intel-sdm.html), [сохраненной копии](https://disk.yandex.ru/i/ppjYD9rcb7v4vQ) или на [удобном сайте](https://www.felixcloutier.com/x86/) 


### Библиотека SimpleIO

Содержит следующие функции:
```
call writei32   // напечатать на экране значение edi
                // как знаковое десятичное число
call writei64   // вывести rdi
call readi32    // ввести с клавиатуры число и сохранить в eax
call readi64    // ввести с клавиатуры число и сохранить в rax
call finish     // завершить исполнение программы
```

Любой вызов подпрограммы сохраняет значения регистров
`rbx, rsp, rbp, r12–r15`. Остальные регистры могут измениться произвольным образом.
