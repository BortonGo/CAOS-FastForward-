### Simple call

Реализуйте функцию вида

```c
long long func(long long x, long long y, long long z);
```

Вычисляющая среднее трех неотрицательных чисел. Считайте что переполнение не возникает. Для деления используйте инструкцию [idiv](https://www.felixcloutier.com/x86/idiv)

В должны сдать только функцию func, функция main будет реализована за вас.

#### Напоминание о System V AMD64 calling convention:
- Первые 6 целочисленных аргументов передаются в регистрах: `%rdi`, `%rsi`, `%rdx`, `%rcx`, `%r8`, `%r9`
- Остальные аргументы передаются через стек (справа налево)
- Возвращаемое значение помещается в `%rax`
- Caller-saved регистры: `%rax`, `%rcx`, `%rdx`, `%rsi`, `%rdi`, `%r8`, `%r9`, `%r10`, `%r11`
- Callee-saved регистры: `%rbx`, `%rbp`, `%r12`, `%r13`, `%r14`, `%r15`
- Если размер аргумента меньше чем размер регистра нужно использовать соответсвующую часть регистра
