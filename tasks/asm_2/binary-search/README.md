### Binary Search

Реализуйте на языке ассемблера функцию `int bsearch40(int *arrs, size_t n)`, возвращающую индекс вхождения числа `40` в массиве `arr`.

Гарантируется, что в массиве `arr`:

- все нечётные числа расположены до чётных,
- нечётные числа упорядочены **по невозрастанию**,
- чётные числа упорядочены **по неубыванию**,
- все числа попарно различны.

Если число `40` отсутствует в массиве `arr`, подпрограмма должна вернуть число `-1`.

Используйте функцию [bsearch](https://man7.org/linux/man-pages/man3/bsearch.3.html) (`man bsearch`).

## Примеры

Вызов

```plain
bsearch40({47, 15, 1, -10, 0, 40, 42}, 7)
```

Результат

```plain
5
```

---

Вызов

```plain
bsearch40({}, 0)
```

Результат

```plain
-1
```
